# EatPlayPay
A Java-based console application that integrates food ordering, billing, multiple payment methods, and mini-games within a centred neon-themed interface using full OOP principles.

**Key Features**

User Sign-Up & Login system

Multi-cuisine food menu

Add to Cart / Remove from Cart / View Cart

Automated billing with GST, discounts, service charges

Payment options: UPI, Card, Wallet, Cash on Delivery

OTP verification + retry logic

Mini-games: Food Riddle, Hangman, Word Puzzle

Neon-style console UI with ANSI colours

Random Order ID generation

Loading animations and smooth console UX


**Technologies Used**

Java

OOP Concepts: Encapsulation, Inheritance, Polymorphism, Abstraction

Java Utilities: Scanner, ArrayList, Random, DecimalFormat

Exception handling

Console UI formatting
